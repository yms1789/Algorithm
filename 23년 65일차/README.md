## 🌀 친구

- 문제 유형(목록): 최단경로, 플로이드-워셜(백준)
- Solution
  - 2-친구 수 여부를 판단하기 위한 `friends`배열을 생성
  - i, j가 서로 친구(`input[i][j] == "Y"`)이거나 i, k가 친구고, k, j가 친구
    (`input[i][k] == "Y" && input[k][j] == "Y"`)라면 i, j가 2-친구 관계이므로
    `friends[i][j] = 1`
  - friends 배열을 반복하면서 각 행에 대해 2-친구 수를 저장해서 가장 2-친구 수가
    큰 경우를 출력
