## 🌀 수들의 합

- 문제 유형(목록): 그리디, 수학(백준)
- 혼자 힘으로 풀었는지 여부: ⭕️
- Solution
  - 서로 다른 N개의 수들을 다 더해서 입력 받은 S를 만들었을 때 N의 최대를 구해야 함
  - 아무래도 N을 최대로 만들기 위해서는 1부터 차례대로 숫자를 증가시켜 나가면서 S에서 빼다가
    S가 증가하고 있는 숫자보다 작아지는 시점에서 중단했을 때가 N이 최대임
  - ex) S가 10이라면
    - $10 - 1 = 9$
    - $9 - 2 = 7$
    - $7 - 3 = 4$
    - $4 - 4 = 0$ → 해당 시점이 마지막!
    - 즉, `N = 4`
