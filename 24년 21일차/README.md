## 🌀 합 구하기

- 문제 유형(목록): 누적합
- 혼자 힘으로 풀었는지 여부: ⭕️
- Solution
  - 처음에는 구간 `i, j`를 받아서
  - arr배열에서 `i, j`에 해당하는 구간 만큼의 값을 더한 후 `result`배열에 넣음
  ➡︎ 시간초과!
  - 해결
    - $a_n = S_n - S_{n- 1}$ 공식을 이용!
    - 미리 각 배열에 0부터 `i`번째 까지의 합을 저장해놓음
    - 이후 구간 수만큼 반복하면서 `i, j`까지의 합 = `0 ~ j까지 합 - 0 ~ i - 1까지의 합`을 이용해서 누적합 값을 `result`에 저장 후 출력
