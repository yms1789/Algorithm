## 🌀 숫자의 표현

- 문제유형(목록): 완전탐색, 정수론(백준)
- Solution
  - n < 3 일 때는 자연수를 연속한 자연수로 표현하는 방법이 1개 밖에 없음
    - 1 => 1, 2 => 2
  - n > 3 이상 일 때
    - sum = 3(1 + 2) 부터 시작
    - 만약, `sum < n` 이라면 `right`를 1씩 증가시켜서 오른쪽 윈도우를 1칸 밂
    - `sum = n`이라면 answer를 1증가 시키고 왼쪽 슬라이딩 윈도우를 1칸 오른쪽으로 당김
    - `sum > n`이라면 answer는 냅두고 왼쪽 슬라이딩 윈도우를 1칸 당김
