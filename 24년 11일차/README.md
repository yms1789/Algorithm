## 🌀 귀여운 라이언

- 문제 유형(목록): 투포인터(백준)
- 혼자 힘으로 풀었는지 여부: ⭕️
- Solution
  - 입력받은 인형을 1, 2로 구분함
  - 시작지점과 끝지점을 0으로 초기화 해놓고 1번(라이언) 인형의 개수가 원하는 K보다 작다면 끝지점을 한칸씩 옮기면서 인형을 구분
  - 1번 인형 개수가 K만큼 됐다면 해당 집합 길이와 기존 최소 길이를 비교해서 저장
  - 가장 작은 연속된 인형 집합을 구해야 하므로 start를 1칸 옮기고 start지점에 있던 인형을 1감소시킴
  - 위 과정을 끝지점이 N까지 갈 때까지 반복
